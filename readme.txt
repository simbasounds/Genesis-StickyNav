Stick your Genesis nav menus to the top on scroll 

== Description ==

Stick your Genesis nav menus to the top on scroll 


== Installation ==

1. Upload 'genesis-sticky-nav' directory to '/wp-content/plugins/'
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How does it work? =

It uses jQuery to modify the CSS of the sticky element when scrolled.
